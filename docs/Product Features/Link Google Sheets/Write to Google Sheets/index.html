Write to Google Sheets

5 minutes readLevel: Beginner Last Updated: June 2026

Write Mode​

When to Use​

Use Write Mode when you need to:

  • Record survey responses in real time.
  • Collect event registrations or sign-ups.
  • Track responses, feedback, or status updates.
  • Store participant information in Google Sheets.
  • Capture data from WhatsApp conversations without manual entry.

Configure Write Mode in Glific​

  1. Log in to your Glific account.

  2. Navigate to Flows → Google Sheets from the left navigation panel.

  3. Click + Create.

  4. Enter the following details:

    • Function: Select Write from the drop-down.
    • Sheet Name: Enter a name for the integration.
    • Google Sheet URL: Open the Google Sheet, copy the URL from the browser, and paste it into the field.

  5. Click Save.

Important

Ensure that the service account has Editor access to the Google Sheet. Without edit permission, Glific will not be able to write data to the sheet.

Configure Write Mode in a Flow​

  1. Create a new flow or open an existing flow.
  2. Add a Link Google Sheet node.
  3. Select the linked Google Sheet from the drop-down.
  4. Choose Write.
  5. Select the worksheet (tab) from the drop-down.
  6. Enter the name of the worksheet where the data should be written (for example, Sheet1).
  7. Specify the starting cell from which Glific should begin writing data (for example, A2).
  8. Under Input data to update in sheet, enter the values to be written.

You can use variables such as:

@results.sheet_input
@contact.name
@contact.fields.age
@contact.phone

Each value is written into a separate column in the same row, starting from the specified cell.

For example, if the starting cell is A2 and you provide:

@contact.name
@contact.phone
@contact.fields.age

The data will be written as:

ABC
NamePhone NumberAge

Write Mode Examples​

Common use cases for Write Mode include:

  • Registration Forms: Save participant names, contact numbers, and event preferences.
  • Attendance Logs: Record participant check-ins directly into a Google Sheet.
  • Survey Responses: Store responses from each participant in a new row.
  • Feedback Collection: Capture ratings, comments, or suggestions automatically.
  • Lead Collection: Save enquiry details received through WhatsApp.
