---

**HSM (Highly Structured Message) templates are pre-approved message formats required for sending outbound messages to contacts outside the 24-hour WhatsApp session window.**

Templates help ensure compliance with WhatsApp's policies and are commonly used to send notifications, alerts, or support messages. They can include placeholders for personalised content and are essential for initiating conversations once the 24-hour session window has expired.

---

**Session Window** — A session window is the 24-hour period following a user's initial interaction with your business on WhatsApp. During this time you can send unlimited messages without additional costs. If the user doesn't respond within that window, you must wait for them to initiate contact again, or use a pre-approved template to reach out.

## When to Use HSM Templates

**Use them for:**

- Initiating conversation with contacts after the 24-hour session window has closed — for example, sending reminders or important follow-ups.
- Broadcasting standardised updates such as activity schedules, weather alerts, crop advisories, appointment confirmations, service announcements, and program events.

## Viewing Your Templates

1. Select **Templates** from the left menu (or go to **Quick tools → Templates**).

![Templates list page](/img/hsm-templates/hsm-templates_list.png)

The **Templates** page lists all your HSM templates, grouped by element name. Each row shows the template name, its language variants, category, and when it was last modified.

Use the filter bar to narrow the list:

- **All Categories** dropdown — filter by Utility, Marketing, or Authentication.
- **All Tags** dropdown — filter by a tag you've assigned to templates.
- **Search bar** — type a template name or element name to find it instantly.

![Filter bar](/img/hsm-templates/hsm-templates_list_filters.png)

**Header actions:**

| Button | What it does |
|---|---|
| **Bulk apply** | Upload a CSV file to submit multiple templates at once |
| **Sync HSM** | Pull the latest approval status from WhatsApp |
| **Template library** | Browse pre-built template suggestions |
| **+ Create** | Open the form to create a new template |

## Creating an HSM Template

1. From the **Templates** page, select **+ Create**.

The **Create HSM Template** page opens with a form on the left and a **Live Preview** panel on the right. The preview updates as you type.

![Create HSM Template page](/img/hsm-templates/hsm-templates_create_form.png)

### Template Details

2. Select a **Language** from the dropdown. This is the primary language for this template. You can add further language variants after the template is approved.

3. Enter an **Element Name** — a unique ID for this template within your WhatsApp Business Account namespace.
   - Use only lowercase letters, numbers, and underscores (e.g., `order_confirmation`).
   - Glific checks availability as you type and shows **Template ID is available** or **Template ID already taken** in real time.

4. Select a **Category** by choosing one of the category cards.

![Category selection cards](/img/hsm-templates/hsm-templates_create_category.png)

   - **Utility** — Account updates, order confirmations, shipping notifications, alerts, and transactional messages.
   - **Marketing** — Promotional content, offers, announcements, product launches, and sales campaigns.
   - **Authentication** — One-time passwords and verification codes for account security.

5. Enter a **Title** — the name for this template as it appears in Glific (e.g., "OTP", "Appointment Reminder", "Activity Preference"). Make it descriptive of the use case.

:::note
Keep **Title** and **Element Name** distinct to avoid mapping issues.
:::

### Message Content

6. Type your message in the **Message Body** field.

![Message body with toolbar](/img/hsm-templates/hsm-templates_create_body.png)

   - Use the toolbar to apply **Bold**, *Italic*, or ~~Strikethrough~~ formatting.
   - Select **+ Add Variable** to insert a placeholder such as `{{1}}`. An example value field appears below the body for each variable — fill these in to see how the final message looks in the preview.
   - Select **AI Assist** to generate message suggestions.
   - The character counter shows your progress toward the 1,024-character limit.

7. Optionally enter a **Footer** — a short subtext of up to 60 characters (e.g., "Powered by YourOrg | Reply STOP to unsubscribe").

### Interactive Buttons (optional)

8. In the **Interactive Buttons** card, select a button type if your template needs one.

![Interactive button type selection](/img/hsm-templates/hsm-templates_create_buttons.png)

   - **Call to Action** — links to a phone number or URL (static or dynamic).
   - **Quick Reply** — lets contacts respond with a single tap (up to 10 buttons).
   - **WhatsApp Form** — links to a WhatsApp-native form.

**Call to Action — URL options:**

| URL type | When to use | Example |
|---|---|---|
| **Static** | Same link for everyone | `https://xyz.org/register` |
| **Dynamic** | Personalised link per contact using `{{1}}` | `https://xyz.org/report/{{1}}` |

:::note
Phone numbers without a country code (e.g., Exotel numbers) cannot be used in Call to Action buttons. Add them in the message body instead.

Emojis are not allowed in button text — the template will be rejected.
:::

### Media Attachment (optional)

9. In the **Media Attachment** card, select an attachment type.

![Media attachment options](/img/hsm-templates/hsm-templates_create_media.png)

   - **Image** — JPG or PNG, max 5 MB.
   - **Document** — PDF, max 16 MB.
   - **Video** — MP4, max 16 MB.

   When you select a type, paste the public URL of the file from your GCS bucket into the **Attachment URL** field. See [Upload files on GCS](https://glific.github.io/docs/docs/Pre%20Onboarding/GCS%20Setup/Upload%20files%20on%20GCS/#upload-media-files-in-the-gcs-bucket) for how to get a public URL.

:::note
Audio and stickers are not supported in HSM templates.

GIFs are not allowed — if you add a GIF as an attachment, the template will be rejected.
:::

### Organisation & Tags (optional)

10. In the **Organisation & Tags** card, select or create a **Tag** to group this template with others. Tags appear as a filter on the **Templates** list page.

### Submit for Approval

11. Select **Submit for Approval**.

The template is sent to WhatsApp for review. Approval can take anywhere from 2 minutes to 48 hours.

12. Return to the **Templates** page and select **Sync HSM** to refresh the status. Once approved, the template status changes to **Approved**.

---

## What if My Template Is Rejected?

- Rejected templates cannot be edited and re-submitted.
- Create a new template to replace the rejected one.

## Can I Edit a Template After Approval?

- No. WhatsApp approval is immutable.
- To make changes, create and submit a new template.

## Approve Multi-lingual HSM Templates

Glific supports applying HSM messages for multiple languages under the same element name. Once the English version is approved, add language variants by linking them to the same element name.

**Step 1: Create and approve the English template**

1. From the **Templates** page, select **+ Create**.
2. Fill in the template in English with a unique element name.
3. Choose a category (e.g., Utility, Marketing).
4. Select **Submit for Approval** and wait for WhatsApp to approve it.

**English**

<img width="519" alt="image" src="https://github.com/user-attachments/assets/29764c5c-80c9-487c-bbe2-857b86c0486b"/>

**Step 2: Add the template in another language (e.g., Hindi)**

1. On the **Templates** page, find the approved English template in the list.
2. Select the **Add new language** icon on the right side of the template row. This opens the Create form with the element name pre-filled.
3. Choose the target language (e.g., Hindi).
4. Enter the translated message in the **Message Body** field.
5. Select **Submit for Approval**.

**Hindi**

<img width="499" alt="image" src="https://github.com/user-attachments/assets/4b76dc4a-ae76-4473-91af-1e1650799c34"/>

**Step 3: Repeat for other languages (optional)**

Follow Step 2 for additional languages (e.g., Marathi, Kannada, Tamil). Always ensure:
- The same element name is used.
- The translation is accurate.
- Placeholders (`{{1}}`, `{{2}}`) match the original.

:::note
When using this template in a flow, add the English template as the default. If a contact selects a different language (e.g., Hindi), Glific automatically sends the Hindi version.
:::

## Bulk Apply HSM Templates

You can submit multiple templates at once using a CSV file.

1. Open this sample [Google Sheet](https://docs.google.com/spreadsheets/d/1x04wI9palh1Ag11TrdBbUj9pnyNOw8-I4834qN3idik/edit#gid=0)

2. Select **File** → **Make a copy** to create your own copy.

![image](https://user-images.githubusercontent.com/40158831/218453349-087e3e8a-bc55-4ce2-9b8a-301a8c0fbefe.jpeg)

3. Fill in the template details in rows. Here is what each column represents:

   1. `Language` — Select the language from the dropdown.
   1. `Title` — The name for your template in Glific (e.g., OTP, Optin, Activity Preference).
   1. `Message` — The template message. Use variables (`{{1}}`) and button syntax (`[Button text, Value]`).
   1. `Sample Message` — Replace variables with actual values to show a complete example.
   1. `Element Name` — The unique template ID for your WABA namespace.
   1. `Category` — `UTILITY`, `MARKETING`, or `AUTHENTICATION`.
   1. `Attachment Type` — `image`, `video`, or `document`.
   1. `Attachment URL` — A public file URL following [Facebook's media guidelines](https://developers.facebook.com/docs/whatsapp/on-premises/reference/media).
   1. `Has Buttons` — `TRUE` or `FALSE`.
   1. `Button Type` — `CALL_TO_ACTION` or `QUICK_REPLY`.
   1. `CTA Button 1 Type` — `Phone Number` or `URL`.
   1. `CTA Button 1 Title` — Button label, e.g., "Call us here".
   1. `CTA Button 1 Value` — Phone number or URL value.
   1. `CTA Button 2 Type`, `CTA Button 2 Title`, `CTA Button 2 Value` — Second CTA button (optional).
   1. `Quick Reply 1 Title` through `Quick Reply 10 Title` — Button labels for up to 10 quick reply buttons.

4. Select **File** → **Download** → **Comma Separated Values (.csv)**.

![image](https://user-images.githubusercontent.com/40158831/218453739-24972a87-534b-4535-a785-76ce4722fee3.jpeg)

5. Go to the **Templates** page and select **Bulk apply** from the top toolbar. Choose the downloaded CSV file.

<img width="1278" alt="Screenshot 2024-03-06 at 3 08 02 PM" src="https://github.com/glific/docs/assets/122982753/4511bd3a-806a-4b5c-8e5b-e68719fe86f2"/>

6. Templates will be processed and a new CSV file with the status of each template will be downloaded automatically.

![image](https://user-images.githubusercontent.com/40158831/218454345-d0e80605-335f-4078-8782-613a96873cae.jpeg)

## HSM Templates Limit

By default, there is a **limit of 250 HSM templates per phone number**. If you need more, submit a request to WhatsApp to increase the limit.

---

## Tips to Get Your Template Approved

1. **Avoid generic messages** — "Hi, here are the activities for today."
1. **Be personal and specific** — "Hi @nameofperson, here are the activities assigned to you. Please complete and submit them by end of day. Thank you."
1. Keeping messages personal and interactive with existing beneficiaries helps approval.
1. GCS attachment links must be **public and accessible** for beneficiaries to open.
1. Use **absolute URLs** in template text. Short URLs cause "Invalid format" rejections.
1. **Emojis are not allowed** in button text.
1. **GIFs are not allowed** as attachments.

:::note
Templates must be in a single language. Do not mix languages within one template (e.g., mixing English and Hindi words in the same body).
:::
